What is jQuery?

jQuery is a fast, small, and feature-rich JavaScript library. It simplifies HTML document traversal, manipulation, event handling, and animation. Its moto is "Write less, do more." U can download this library or use CDN servers to include it in your web pages.

jQuery is JS itself. It is not a programming language. It is a library built with JS to make it easier to use.
