Arrays in JavaScript

  1. Arrays in JS are more like collections.
  2. Its Dynamic, U can add, remove elements at any point.
  3. All Arrays in JS are objects of Array class.
  4. Technically they are all objects.
  5. With Array functions, we can add, remove, update and search the elements of the array
  6. Arrays in JS behaves more like LIST of Java
  7. push, unshift and splice are used to manipulate the data of the array.
