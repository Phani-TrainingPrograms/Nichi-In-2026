ES 6 Features

List of feature

  1. Arrow Functions : Lambda Functions that reduces the complexity of Anonymous functions
  2. Template Literals: Interpollation of strings.
  3. Destructuring: Breaking the object to local variables is called destructing.
  4. Default Parameters: It shall be the last of the parameter list of the function.
  5. Spread Operator: Used to expand arrays and objects into individual elements.
  6. Rest Parameters: Used to capture an indefinite number of arguments as an array.
  7. Let and Const: Scoped based variables and consts are values that are not reassignable.
