# What is Express?
- Express is a library for creating web apps and services in a faster and relieable manner.
- It supports app development for web, mvc, services and socket programming on web. 
- Its an open source package available thru NPM. 
- NPM is a repository of all open source NODE Packages that can be consumed by the Nodejs Developers. 
- Its an open repository where developers contribute new packages for the community. 

# How to install Express
```
npm install express.
```
